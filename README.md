# Stellar Activity Taxonomy

This repository contains the data and source code (analysis pipeline) used in the study on stellar activity taxonomy. The main objective of this project is to analyze and classify different types of stellar activities based on physical indices and parameters.

The repository is organized as follows:

* **`principal.ipynb`**: The main notebook containing the entire data analysis flow. It includes:
    * Data loading and processing.
    * Calculation of statistics and correlations.
    * Generation of plots and visualizations presented in the study.
* **`table2.csv`, `table3.csv`, `table4.csv`**: CSV data files containing stellar parameters and activity indices. These files correspond to the tables presented in the original publication/work.
* **`figures/`**: Directory dedicated to storing figures generated by the notebook.
